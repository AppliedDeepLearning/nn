# Datasets

Datasets are available under `nn.datasets`.


## Usage

To use a dataset, load its data using:

```py
(x_train, y_train), (x_test, y_test) = nn.datasets.mnist.load_data()
```


## Available Datasets

- `boston_housing`
- `cifar10`
- `cifar100`
- `fashion_mnist`
- `imdb`
- `mnist`
- `reuters`
